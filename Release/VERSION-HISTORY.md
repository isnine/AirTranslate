# AirTranslate Version History

## 1.2.0 - 2026-05-13

Release focus: GPT realtime captions, multilingual documentation, and App Store release preparation.

### Highlights

- Optional GPT realtime transcription and translation modes.
- Realtime translation-only path with optional translated audio playback.
- Floating captions in GPT mode show the current live caption unit instead of the accumulated transcript.
- User-provided OpenAI API keys are stored in macOS Keychain and are not bundled with the app.
- App Store release kit with sandbox entitlements, packaging script, screenshot assets, privacy notice, and App Store Connect draft metadata.
- English, Korean, Japanese, and Simplified Chinese README files.
- Apache 2.0 source license and NOTICE attribution.

### Release Checks

- `swift build`
- `swift test`
- Secret-pattern scan for API keys, bearer tokens, private keys, and local secret files.

### Packaging Notes

- Default version metadata: `VERSION=1.2.0`, `BUILD_NUMBER=120`.
- Local validation output goes to `Release/product/`.
- `Release/product/` is generated output and should not be committed.

## 1.1.0 - 2026-05-10

Release focus: live audio meter and clearer header controls.

### Highlights

- Centered live audio waveform meter.
- Improved hover, pressed, active, and click-confirmation feedback for transport controls.
- Version metadata for local app bundles generated by `script/build_and_run.sh`.

### Release Checks

- `swift build`
- `git diff --check`
- Local app-bundle launch check.

## 2026-05-09 - Library Modal UI

Release focus: saved transcript management.

### Highlights

- Saved transcript management moved from the sidebar into a focused modal.
- Original-plus-translation saved transcripts grouped into a single library row.
- Confirmation-protected delete-all action.

## 2026-05-09 - Transcript Control and Stability

Release focus: long-session stability and transcript readability.

### Highlights

- Configurable silence interval for paragraph breaks.
- Bounded speech analyzer input buffering.
- Bounded translation segment cache.
- Reduced text animation work for long transcript updates.
